Nightcrawler

Backend Focus Development: Python, PostgreSQL, Django

Nightcrawler (http://rodeur.herokuapp.com) is a web application that collects and analyzes news articles from different countries. The application provides analysis on sentiment analysis and top 10 frequent words.
Nightcrawler also displays current international state of affairs by finding keywords (name) of other countries in the selected country. You are also able to put your own keyword to see how news articles of each countries react to the keyword.
The application is powered by Python, Django and PostgreSQL and many Python libraries.
Check out the code in my GitHub.

Piano Simulator

School Project: MIPS, Mars MIPS Simulator

Piano Simulator is a MIPS code that I made on my first year learning computer programming as a school project. The programming code has a record feature that users can provide their own piece of music; the code also has a play feature that users can play to match the falling musical notes of user-provided music.
Piano Simulator is supposed to provide users feeling that they are playing a real piano through computer keyboard, since the position of keyboard keys matches to the position of real piano in terms of musical notes.
